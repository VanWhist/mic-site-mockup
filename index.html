Mission / Vision / Value

MICに携わるすべての人が、
モーグルに関われて良かったと思える場所を。

01 — Mission

関わって良かったと、
心から思える場所に。

選手、コーチ、保護者、支えてくれる企業様。MICに携わるすべての方が、モーグルに関われて良かったと思えるチームを目指します。

02 — Vision

今日より明日、
今週より来週、上達する。

大きな飛躍だけを追わない。今日より明日、今週より来週、今月より来月。着実な積み重ねが、いつか大きな結果になると信じています。

03 — Value

全身全霊で、
上達をサポートする。

雪上練習、大会引率、ウォータージャンプ、トランポリン。年間100日以上のプログラムで、選手一人ひとりの成長を全力で支えます。
